ğŸš€ Getting Started

Prerequisites

  • Flutter SDK installed
  • Dart SDK (included with Flutter)
  • Android Studio or Xcode (for device/emulator)
  • Supabase project (URL + anon/public key)

Installation

# Clone the repository
git clone https://github.com/SwapnilChand0623/Campus-Roommate.-Finder.git
cd Campus-Roommate.-Finder

# Install dependencies
flutter pub get

Configuration

Create a config file lib/config/supabase_config.dart:

class SupabaseConfig {
  static const String url = '<YOUR_SUPABASE_URL>';
  static const String anonKey = '<YOUR_SUPABASE_ANON_KEY>';
}

Database Setup

  • Run the provided SQL migrations in the Supabase SQL editor
  • Create storage buckets for profile photos and listings
  • Apply Row Level Security policies

Run the App

flutter run
