📊 Stockiez – AI Stock Advisor

AI-powered stock analysis & buy/sell recommendations

🧠 What is Stockiez?

Stockiez is a full-stack stock prediction platform built with a Flask backend and React frontend. It uses LSTM neural networks to forecast future prices for select stocks and delivers AI-based buy/sell recommendations based on model confidence and historical trend analysis.

✨ Features

  • 📈 Predicts SPY, AAPL, MSFT, NKE
  • 🔄 Real-time stock data via yfinance
  • 🧠 LSTM model trained on 60-day price windows
  • 📊 Interactive frontend with charts and recommendations
  • 🧪 Model accuracy & confidence rating displayed

🧰 Tech Stack

  • Frontend: React, Chart.js, Axios
  • Backend: Python, Flask, TensorFlow, yfinance
  • Model: LSTM (Long Short-Term Memory)
